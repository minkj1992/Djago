# Django
> 파이썬으로 작성된 오픈 소스 웹 어플리케이션 프레임워크

## 프레임워크
- 웹 서비스를 만들때마다 반복해서 만들어야 하는 것들을 표준화 하여 묶어 놓은 개발환경
- 쉽게 말해, 웹프로그램을 만들기 위해 사용하는 손쉬운 도구라고 생각하면 된다.

## 프레임 워크 vs 라이브러리
- 공통점
    - 개발에서 반복되는 작업을 줄여주는 역할
- 차이점
    - 프레임워크: 사용할 때 정해진 방식대로 코드를 짜도록 강요받는다.
    - 라이브러리: 그때 그때 필요할때 가져다 쓸수 있다.
